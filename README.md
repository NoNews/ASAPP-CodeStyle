## XML

### layout names:
  - RecyclerView :  `item_name`
  - Activity : `activity_name`
  - Fragment: `fragment_name`
  - Animation: `anim_name`
  - Menu: `menu_name`
  



### View elements

  - TextView :  `tv_name`
  - ImageView: `iv_name`
  - Button: `btn_name`
  - CircleImageView: `civ_name`
  - CardView: `cv_name`
  - LinearLayout, FrameLayout, RelativeLayout, AbsoluteLayout, CoordinatorLayout: `lt_name`
